# 🥖 مخبز الوليد (El Walid Bakery)

موقع إلكتروني تعريفي يعرض منتجات مخبز الوليد من المخبوزات الطازجة والقرص الطرية وكحك العيد، مصمم لتوفير تجربة مستخدم سريعة وسلسة على مختلف الأجهزة.

A responsive, high-performance showcase website built for **El Walid Bakery**, featuring a modern interface to display fresh baked goods, traditional pastries, and seasonal treats.

---

## 🚀 Live Demo / الرابط المباشر
You can view the live website here: 
🔗 **[[elwalid-bakery.vercel.app](https://alwaleed-bakery.vercel.app/)]**

---

## ✨ Features / المميزات

* **Responsive Design:** Fully optimized for mobile phones, tablets, and desktop viewports.
* **Fast Loading:** Optimized image handling and assets served via Vercel's global CDN network.
* **Modern UI:** Clean, elegant visual layout showcasing bakery items with clear descriptions.
* **Arabic & Custom Typography:** Styled using premium Arabic web fonts matching the brand identity.

---

## 🛠️ Built With / التقنيات المستخدمة

* **Frontend:** HTML5, CSS3, JavaScript
* **Development Tool:** Google Antigravity 2.0
* **Hosting & Deployment:** Vercel
